## 5.3 Communication Plan

To ensure smooth collaboration, Team SPICA adheres to the following communication cadence:

* **Daily Stand-up:** 15 minutes every morning to discuss progress and blockers.
* **Weekly Sprint Review:** Every Friday to demonstrate completed features to the Mentor.
* **Tools:**
    * **Source Control:** GitHub (Monorepo).
    * **Task Management:** Trello / Jira / GitHub Projects.
    * **Communication:** Slack / Discord / Microsoft Teams.
