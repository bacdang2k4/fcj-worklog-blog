**Trong trang này** bạn sẽ cần giới thiệu worklog của bạn **như thế nào**? Bạn hoàn thành chương trình trong vòng **bao nhiêu tuần**? Bạn **đã làm gì** trong các tuần đó?


Thông thường và cũng là tiêu chuẩn, một worklog được thực hiện trong khoảng 3 tháng (trong suốt thời gian thực tập) với nội dung các tuần như sau:

**Tuần 1:** [Làm quen với AWS và các dịch vụ cơ bản trong AWS](1.1-week1/)

**Tuần 2:** [Tìm hiểu về VPC trong AWS](1.2-week2/)

**Tuần 3:** [Nắm vững các kiến thức cơ bản về EC2](1.3-week3/)

**Tuần 4:** [Tìm hiểu các dịch vụ lưu trữ của AWS và làm quen với hệ thống lệnh AWS CLI](1.4-week4/)

**Tuần 5:** [Học và thực hành các nội dung về bảo mật AWS (IAM, Cognito, Organization, Identity Center, KMS, Security Hub)](1.5-week5/)

**Tuần 6:** [Học Microservice Architecture, tìm hiểu gRPC và Apache Kafka, ôn tập cho giữa kỳ và hoàn thành bản vẽ kiến trúc hệ thống project nhóm](1.6-week6/)

**Tuần 7:** [Học AWS Lambda, cài đặt và sử dụng AWS CLI, thực hành deploy Java function lên Lambda và xây dựng API với API Gateway](1.7-week7/)

**Tuần 8:** [Hoàn thành proposal nhóm, ôn tập toàn diện kiến thức AWS (bảo mật, kiến trúc, hiệu năng, chi phí) và tham gia kỳ thi giữa kỳ](1.8-week8/)

**Tuần 9:** [Tìm hiểu AWS IoT Core và AWS Amplify để mở rộng kiến thức về hệ sinh thái AWS và phục vụ cho project](1.9-week9/)

**Tuần 10:** [Học Terraform, GitHub Actions và triển khai cấu trúc hạ tầng cho dự án](1.10-week10/)

**Tuần 11:** [Triển khai toàn bộ hạ tầng AWS bằng Terraform và xây dựng pipeline GitHub Actions cho hệ thống](1.11-week11/)

**Tuần 12:** [Hoàn thành demo dự án nhóm, chuẩn bị slide thuyết trình và tổng hợp worklog cá nhân](1.12-week12/)
